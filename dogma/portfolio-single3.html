Sistema de control de inventario de insumos

Se creó un sistema para la empresa “La churrería” en Ciudad Victoria Tamaulipas el cual tuvo el fin de solucionar el problema con el que contaba la dueña a la hora de realizar un conteo sobre los insumos que se requieren para preparar su producto, así mismo como también tenía la necesidad de resolver problemas contables con falta de materia prima y los productos desechables. Se terminó con el sistema satisfactoriamente y actualmente se encuentra en uso.
